# QMail 

- **Backend:** NestJS + SQLite 
- **Frontend:** Vue 3 + Quasar + Pinia + Axios

### 1. Backend

```bash
# development (3000 порт по умолчанию)
cd backend
npm install
npm run start:dev
```

### 2. Frontend

```bash
# development (3001 порт по умолчанию, фронтенд делает запросы на http://localhost:3000)
cd frontend
npm install
npm run dev
```
